Forms

Angular supports two methods for managing forms;

  • Template-driven forms
  • Reactive forms

Template-driven forms

Relies on directives in the template to create and manipulate the underlying object model.

They don't scale well and are best suited for simple forms. i.e. login, newsletter subscriptions etc.

Setup form model

Template-driven forms use NgModel directives to handle the form model.

{{templateDrivenModel}}

Validation

Tied to template directives, and must provide custom validator directives that wrap validation functions.

{{templateDrivenValidation}}

For details on creating customValidatorDirective refer to the official documentation

Reactive forms

Provide direct, explicit access to the underlying form's object model

Setup form model

You can define the form model in the component class using FormControl directive.

{{reactiveModel}}

Validation

Define custom validators as functions that receive a control to validate

{{reactiveValidation}}
